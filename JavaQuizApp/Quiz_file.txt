Section A: Arrays Quiz (10 MCQs)

Question 1
What is an array in Java?
A. A collection of different data types
B. A collection of elements of the same type
C. A loop structure
D. A conditional statement
Correct Answer: B
Question 2
What is the index of the first element in an array?
A. 1
B. -1
C. 0
D. 10
Correct Answer: C
Question 3
Which symbol is used to declare an array in Java?
A. ()
B. {}
C. []
D. <>
Correct Answer: C
Question 4
Which of the following correctly declares an integer array?
A. int arr[];
B. array int arr;
C. arr int[];
D. int (arr);
Correct Answer: A
Question 5
What will be the length of the following array? int arr[] = new int[5];
A. 4
B. 5
C. 6
D. 10
Correct Answer: B
Question 6
Which method/property is used to get the length of an array?
A. length()
B. size()
C. length
D. getLength()
Correct Answer: C
Question 7
Which loop is commonly used to access array elements?
A. while loop
B. for loop
C. do-while loop
D. All of the above
Correct Answer: D
Question 8
What happens if you access an index outside the array size?
A. Program stops
B. ArrayIndexOutOfBoundsException occurs
C. Compiler error
D. Nothing happens
Correct Answer: B
Question 9
Which keyword is used to create an array object?
A. create
B. new
C. make
D. array
Correct Answer: B
Question 10
Arrays in Java are stored in:
A. Stack memory
B. Heap memory
C. CPU register
D. Disk memory
Correct Answer: B
